Projects

Djikstra
Using JavaScript, I created a small demo of Dijkstra's shortest path algorithm. The program allows the user to input a start and end node, and the algorithm finds the shortest path between them. It's a great way to visualize how Dijkstra's algorithm works in practice. It also allow user to enter a weight value for the algorithm to act like A*.

Raycasting 2D
I created a small 2D raycasting engine project inspired by the 1993 game Doom. Using Javascript, I implemented a basic version of the algorithm used to render the walls. By casting rays from the player's position and calculating the distance to each wall, I was able to create a basic 3D environment that the player can explore.

Bezier Curve Generator
I built a small Bezier curve generator using JavaScript. The program allows users to add and manipulate control points to create smooth and curved paths. It's a simple but effective tool for creating custom curves, and a great way to learn about the math behind Bezier curves.

Image Processing
A collection of small yet interesting projects that utilize OpenCV and Python for image processing. These projects cover a wide range of topics, from contour detection to image correction and image enhancement. Each project highlights my skills in computer vision, Python programming, and OpenCV libraries, demonstrating my ability to deliver high-quality results in image processing.

Raytracing MPI
I implemented an MPI-based raytracing tool using C to render simple 3D scenes in parallel. By leveraging the MPI message-passing interface, our project significantly reduced rendering time by enabling multiple processors to work simultaneously.

Corewar
The CoreWar game implemented in Java is a small project that showcases the basic principles of the game. Players write assembly programs and pit them against each other in a virtual memory arena. The last program running wins.

Card Cryptography
A small cryptography project that leverages a deck of 52 cards and JavaScript programming. The project implements a unique cryptographic algorithm based on shuffling the deck of cards and using them to encrypt and decrypt messages.

Projet 3D
